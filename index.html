Especialização e competências técnicas

Desenvolvedor de software com formação em Análise e Desenvolvimento de Sistemas pela Faculdade SENAC Florianópolis, especializado no desenvolvimento de aplicações enterprise com arquitetura robusta e escalável.

Backend: Desenvolvo APIs RESTful utilizando Java Spring Boot, implementando autenticação JWT, segurança SSL/TLS, e integração com bancos relacionais MySQL. Experiência com Spring Security, JPA/Hibernate, Jakarta Validation e implementação de padrões de projeto para aplicações de alta performance.

Frontend: Desenvolvimento de interfaces responsivas com Angular, utilizando componentes standalone, formulários reativos, e integração com serviços backend. Implementação de arquitetura modular e reutilizável seguindo as melhores práticas do framework.

Mobile: Desenvolvimento de aplicações nativas multiplataforma com React Native e Expo, incluindo planejamento de arquitetura, implementação de funcionalidades complexas e preparação para publicação nas lojas de aplicativos.

Experiência Profissional: Atuação em suporte técnico na Prefeitura Municipal de São José, desenvolvendo habilidades de resolução de problemas, comunicação técnica e atendimento ao usuário final.
